# Portafolio CV - Gerardo Rodríguez Pérez

Portafolio web interactivo desarrollado para presentar mi perfil profesional, formación académica, proyectos, habilidades técnicas y CV descargable.

El objetivo de este sitio es funcionar como una extensión digital de mi currículum, mostrando de forma visual e interactiva mi perfil como estudiante de Matemáticas Aplicadas y Computación con interés en desarrollo de software, análisis de datos y tecnologías de la información.

## Vista general

El sitio incluye:

- Sección principal con presentación profesional.
- CV interactivo organizado por pestañas.
- Proyectos destacados con filtros y ventanas de detalle.
- Sección de habilidades técnicas.
- Información de contacto.
- Botón para descargar el CV en PDF.
- Diseño responsive para escritorio y dispositivos móviles.
- Fondo animado con gradientes y componentes tipo glassmorphism.

## Tecnologías utilizadas

Presentación del perfil profesional, enfoque de carrera y accesos rápidos a proyectos, GitHub y CV.

Proyectos académicos y personales relacionados con desarrollo web, backend, automatización y manejo de información.

## Proyectos destacados

### StayFlow

API y aplicación de reservas de alojamientos, desarrollada como proyecto universitario con enfoque en una estructura cercana a producción.

Tecnologías: Java, Spring Boot, MySQL, Postman, Swagger y GitHub.

### Wide Receiver

Chatbot web sobre reglas de la NFL, desarrollado para responder preguntas manteniendo enfoque temático y consistencia en las respuestas.

Tecnologías: Typebot, n8n, Pinecone y API Keys.

### NutriAcatlán

Página web básica enfocada en apoyo nutricional, cálculo de calorías, macronutrientes y generación de planes según necesidades del usuario.

Tecnologías: HTML, CSS y JavaScript.
